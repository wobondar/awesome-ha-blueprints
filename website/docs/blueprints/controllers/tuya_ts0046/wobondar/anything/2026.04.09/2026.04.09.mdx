## Description

This blueprint provides universal support for running any custom action when a button is pressed on the provided Tuya TS0046 Wireless switch with 6 buttons. Supports controllers integrated with Zigbee2MQTT.

The remote natively supports single press, double press, and hold events for each of its 6 buttons, giving you 18 configurable actions in total.

:::note
This blueprint is based on the [Moes XH-SY-04Z blueprint](/docs/blueprints/controllers/moes_xh_sy_04z) by [@wobondar](https://github.com/wobondar).
:::

:::tip
Automations created with this blueprint is not connected to any [Hooks](/docs/blueprints/hooks).
:::
